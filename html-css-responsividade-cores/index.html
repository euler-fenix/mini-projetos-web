HTML

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas web. Ela é composta por uma série de elementos que permitem inserir e organizar conteúdo, como texto, imagens, links, tabelas, e muito mais. Cada elemento HTML é representado por uma tag, e essas tags são utilizadas para definir diferentes partes de uma página web.

Por exemplo, a tag <p> é usada para criar parágrafos, enquanto a tag <a> é usada para criar links. O HTML é uma linguagem de marcação, o que significa que ele descreve a estrutura do conteúdo na web, mas não dita como ele deve ser estilizado ou apresentado. Para a estilização, usamos CSS (Cascading Style Sheets), que pode ser combinado com HTML para criar páginas web visuais e interativas.

    <p>parágrafo de exemplo em HTML.</p>
    <a href="https://example.com">Este é um link</a>

</html>

CSS

CSS (Cascading Style Sheets) é a linguagem usada para descrever a apresentação de documentos HTML. Ele controla a aparência dos elementos da página, incluindo layouts, cores, fontes e espaçamentos. Com CSS, você pode criar designs visuais atraentes e responsivos que se adaptam a diferentes dispositivos e tamanhos de tela.

CSS funciona com seletores e declarações. Seletores são utilizados para selecionar os elementos HTML que você deseja estilizar. Declarações são compostas por propriedades e valores que definem o estilo a ser aplicado. Por exemplo, para definir a cor do texto de um parágrafo, você pode usar o seletor <p> e a declaração color: blue;. CSS também permite o uso de variáveis para gerenciar paletas de cores e outros valores reutilizáveis de maneira eficiente.

▼ Exemplo de Codigo CSS 

body {
    font-family: Arial, sans-serif;
    background-color: #f0f8ff;

    padding: 0;
}

    text-align: center;
}
p {
    margin: 10px 0;
}
a {
    color: #0056b3;
    text-decoration: none;
}
a:hover {
    text-decoration: underline;
}
